ABOUT

What runs under the hood

A coordinated team of LLM agents wraps RDKit + ADMET-AI behind a FastAPI service.

Multi-Agent Architecture

PrismBB Drug is composed of four cooperative agents orchestrated by Agno. Each agent owns a single responsibility and exposes typed tool calls so the analysis pipeline is reproducible and inspectable. A fifth pipeline layer runs AutoDock Vina for protein-ligand docking.

P

ParserAgent

RDKit-based SMILES sanitization with 17+ descriptors, InChI, and Lipinski checks.

C

ConformerAgent

Generates 3D conformers using UFF or MMFF force fields and outputs PDB blocks.

A

ADMETAgent

Predicts absorption, distribution, metabolism, excretion, and toxicity properties.

R

RenderAgent

Aggregates outputs into a single structured payload consumed by the UI.

Highlighted capabilities

  • SMILES parsing & validation
  • InChI / InChIKey generation
  • TPSA, LogP, HBD, HBA, rotatable bonds
  • Lipinski violations and drug-likeness
  • 3D conformer embedding (UFF / MMFF)
  • ADMET property prediction
  • Live 3D molecular visualization
  • SDF · PDB · MOL · XYZ uploads

Technology stack

  • FastAPI + Pydantic
  • Agno multi-agent framework
  • Groq LLaMA 3.1 70B
  • RDKit (cheminformatics)
  • ADMET-AI (property models)
  • 3Dmol.js (WebGL viewer)
  • Flask (proxy + UI)
  • Vanilla ES modules — zero npm bloat
